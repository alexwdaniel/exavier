# Exavier

An Elixir Slackbot example

#### Setup

Install Erlang and Elixir:

    $ brew install erlang
    $ brew install elixir

Install project dependencies:

    $ mix deps.get

Set your `SLACK_API_TOKEN` environment variable

Run with:

    $ mix run --no-halt

#### Tests

To run ExUnit tests:

    $ mix test
